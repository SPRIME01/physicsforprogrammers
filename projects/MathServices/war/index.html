Math Services on Google Appengine

Utility math routines from a google appengine application. This service also acts as an experiment in literate programming. Source for all of the examples are available. (version 0.0.2)

Available Applications:
Basic Math Utilities
  • Range - Print the range of numbers 'from', 'to'.
  • Fibonacci - Find the fibonacci number. Change the parameter 'n' in your browser address bar. / (literate example)
  • Fibonacci numbers - Print the first set of fibonacci numbers. Change the parameter 'n' in your browser address bar.
  • Summation - sum(n). Change the parameter 'n' in your browser address bar.
  • Add two numbers - plus(x, y). Change the arguments 'a' and 'b'.
  • Numbers Database - Interesting numbers.
  • Dates Database - Interesting dates.
  • Mandelbrot Fractal - Image of the mandelbrot fractal (10-15 second render time)
